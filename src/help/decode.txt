------------------------------------ Decode ------------------------------------

Decode any supporting FrAD files to another format. It highly
leans on ffmpeg for re-encoding.

Usage: {frad} decode <path/to/audio.frad> [kwargs...]

Options:

    --ecc
        Check and fix errors (default: false, alias: e, enable-ecc)
    --gain [<gain> dBFS | <amplitude>]
        Gain level in dBFS or Amplitude (alias: g)

    --output <path/to/audio.pcm>
        Output file path (alias: o, out)
    
    --codec <str>
        Codec for decoding, default: 24-bit FLAC

    --bits <int>
        Bit depth for lossless output (default: 24, alias: b, bit)
    --quality <int>
        Quality for lossy output in <bitrate>[c|a] or <quality>v (alias: q)
    --sample-rate <int>
        New sample rate of output file (alias: sr, srate)

    --verbose
        Verbose output (alias: v)

    --ffmpeg [str]
        Pass a custom FFmpeg command for decoding.
        Recommended for advanced users. Any options specified after --ffmpeg
            will be passed directly to FFmpeg.
        Output file name auto-detection not supported.
        (alias: ff, directcmd, direct-cmd, direct-ffmpeg)
